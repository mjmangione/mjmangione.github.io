A Few Projects

Below lies some coursework which emphasizes some of my interests and skills. Click for more information.

Rock & Roll and the Gabor Transform

Analysis of popular Rock songs using the Gabor Transform, a time-specific sweep of the Fast Fourier Transform algorithm which locates the individual frequencies of the song in time.

Classifying MNIST digits using Linear Discriminant Analysis

This project classifies the 10 digits of the MNIST dataset by finding the optimal basis which distinguishes each category. This is achieved using the singular value decomposition of the dataset.

Bird Classification using Transfer Learning

With the goal of classifying 555 different species of birds using machine learning, this project imports large pre-trained convolutional neural network architectures, and then further trains them using our specific bird image dataset.

Bose-Einstein Condensate 3D Simulation

In order to simulate the Bose-Einstein Condensate, an iterative numerical algorithm is used to approximate the Gross-Pitaevskii partial differenatil equation, a derivative of the Schrödinger equation.

Garduino

I used Arduino to build a machine to care for my plants while I was away. It controls 3 water pumps on separate timers, and has a LED User Interface based on three buttons that can turn on the pumps and alter the timers.

Hydroponic Garden

I created an automated hydroponic system for sustaining plant life indoors. It uses a water pump, an aerator, food-safe PVC piping and automated overhead lighting. It was designed to grow as much food possible in the least amount of floor space, supporting plants from seedlings to maturity. It is best suited for dense plants like lettuce, bok choy, and strawberries.
